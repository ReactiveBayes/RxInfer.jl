
# Contributing to RxInfer

**RxInfer** is a community-driven project and we welcome contributions of all kinds!

To get started:

* Check out the **Contributing Guide**: https://docs.rxinfer.com/stable/contributing/guide
* Review the **Contributing Guidelines**: https://docs.rxinfer.com/stable/contributing/guidelines
* Browse **beginner-friendly issues** to find something that interests you:
  https://github.com/ReactiveBayes/RxInfer.jl/issues?q=is%3Aopen+is%3Aissue+label%3A%22good+first+issue%22

We welcome contributions such as:

* Bug reports and fixes
* Feature suggestions
* Documentation improvements
* Tests and examples
* Performance improvements

## Contributing to the RxInfer Ecosystem

The RxInfer ecosystem consists of several closely related packages. Contributions to these projects are also very welcome:

- GraphPPL — https://github.com/ReactiveBayes/GraphPPL.jl
- ReactiveMP — https://github.com/ReactiveBayes/ReactiveMP.jl
- ExponentialFamily — https://github.com/ReactiveBayes/ExponentialFamily.jl
- Browse other libraries in the ecosystem - https://github.com/ReactiveBayes

Many improvements and new features involve collaboration across these repositories, so feel free to explore and contribute wherever your interests lie.

If you're unsure where to start, the *good first issue* label is a great place to begin.
Feel free to open an issue or start a discussion if you have questions.

Thank you for helping improve RxInfer! 🚀
